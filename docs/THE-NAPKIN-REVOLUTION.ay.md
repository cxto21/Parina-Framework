## Mä amta software sustainable ukat resilient uñstayañataki

Nayraqataxa, pampachapxita. Janiw phiñasiri uñstayañ munkiti, janiw uñstayasiñamatak suykti, ukatx Veronica jach'a yatiñani sarataparjamaxa, lurayawinakamax walikiwa.

Wali jaya pacha qillqañ taypin qillqaspa code. Kunawsatix lurañ pachar mistuwayta, aski lurawinaka katuqañ amtawayta, jan 10 pantjasiwinaka lurañataki, ukatx ukhamaw wali pacha uñjasiwayta.

Akapacharuw web lurañ amtawayta, ukampis walipuni qarisiwayta sapa kuti qillqaña, sapa kuti kipka yatiñanaka apnaqaña, ukatx kunatix oficinan lurañäki uk kipka luraña. Machaq kuns thaqhayäta, jan ch'aman thakhi.

Jichhax khititix mayki ukax katuqiwa: wali aski lurayawinaka jikxattawayta, code generadores lurañ phaxsinx wali kusiskañäna, mä sintonía, mä k'ajkir nina uñstayasa.

Ukatx academia ukar kutt'añ amtawayta machaq jatha jaqinakar yatiyañataki ukham lurañanakax utjiwa... walja kutiw jawillt'apxitu... naya markat anqäxansa.

Ukampis castilla aruta uñt'at jach'a jawillt'awi ukarus chikañchasiwayta, Youtube ukarus videonaka uñt'ayawayta, ukatx pacha churawayta machaq apnaqirinakar ch'amanchañataki, khitinakatix lurañatak programmed, thaqhir jaqinaka.

Walja qillqatanak qillqawayta, ukampis yatichiri amtawix utjaskaspachänwa: machaq apnaqirinakar uñt'ayañ munkta kunjams llunk'utanakax lurasi, kunjams diseñasi, kunas ukanakan utji.

Janiw chikañchasipkti ukatx nayapuni thakhinkawayta. Walja llunk'utanaka lurañ amtawayta uka jach'a apnaqirinakjam, ukampis janiw wali askïkiti. Jupanakax wali jaya pacha nayrankapxatayna.

Thakhit samart'asa, mä urux modern PHP ukat SOLID patrónt'at yatiqawi alawayta, ukatx arquitectura, CQRS patrónt'at qillqatanaka uñch'ukawayta, ukatx uñjasiwayta qhananchat qillqata (maintainable code).

Ukatx juk'amp phiskhu lurañ amtawayta, mä ch'usa amtar kutt'añataki, ukhamat khitis p'iqipar uchasiñapataki.

Taqi kuns lurañ taypin, amuyux qhananchawayi. Taqi kunatix lurañäki ukax qillqa (texto) procesar lurawikiwa. ¿Machaqati? Unix pachat jutañäni ukax taqiniw amuyapxi. Ukampis taqpach jaqinaka, thakhin sarnaqirinaka, khitinakatix machaq milagro yatiyawinak katuqapxi... ¿amuyapxiti?

Ukhamatwa amta utjawayi mä taqpach framework mä jisk'a servilleta papel ukhar uchañataki.

Ukampis janiw anatt'añakiti, Secure Code Warrior yatiqawix p'iqin wali manqharuw katuqawayi. Taqpach uñjataram seguro, aski k'acha, yant'ataram (testeable) ukatx sinti amuykay ukat extensible uñt'ata.

Jisk'a diagramataki sinti jark'awinakawa, ¿janicha?

Nayriri lurawi, mayirinakar p'iqiñchiri: nayriri p'iqiñchiri (front controller). Machaq sutiniwa, ukampis servidor ukar taqi jutirinak kipka qillqatar (index.php) khitañakiwa.

Waliki, taqpach jutirinakax p'iqiñchiri ukaruw jutapxi.

Payïri lurawi, sapa mayiwix mä sapa lurawiniwa (SOLID ukan S k'apak lurawi), ukatx sinti phiskhuchawaytwa: mä handler (luriri). Ukakiwa.

Juk'a samart'ata, handler-ax mayiw jutiri (request) katuqi, procesi ukat kutt'ayi (texto), ukhamax kutt'ayawip (response) uñstayi.

Ukatx amtaw taypin... ¡ukakiwa!

¡Ukampis Nelson! Seguridad, extensibilidad, aski qillqata ukat yant'atäñap yatichawayta!

SOLID ukan amtawix Interfaces apnaqañawa, ukanax jan luräni planonakäspas ukhamawa, ukampis contratos (métodos) utji, ukat apnaqirinakax phuqañapawa (implementar).

¿Kunatakis interfaz ukaxa? Jisk'aki, naya uñjatapxarux, aski uñstatäna kipka lurawinakar uñstayasiñataki, ukampis mayjjt'at phuqhawinampi (polimorfismo).

¿Ukatx interfaces kunas churi? ¡Extensibilidad ukat estandarización ch'amanchaña!

Jichhax utjiwa aski p'iqiñchirinaka. Thakhi apnaqir thaqht'aña (Router). Jutirinakar mayiwinak thakhinakar khitañataki jan ukax pantjat yatiyawinak uñstayañataki (despachador) ukax Kernel sutiyawayta.

Kunawsatix .NET, Java jan ukax JSP luratächi, Request ukat Response apnaqaspa, ukanax 'apnaqiri jaqin mayiwinapa' ukat 'kutt'ayapxatanas' ukhamawa.

Mayiwinakat uñachawinaka:
- Mä jisk'a link
- Mä link parámetros-naka
- Mä formulario qillqatanakampi
- Mä formulario uñtatampi

Kutt'ayawit uñachawinaka:
- Mä status code
- Jan aski qillqata
- HTML
- JSON
- XML
- Qillqat uñtatanaka
- Mä Redirección
- etc., etc., etc.

¿Ukatx seguridad?
Mä urux amuyawayta mayiw jutpanki, thakhi jark'irinakax amuyapxañapawa luraspa janicha. Ukanax palacio guardianawa, software arutxa, Middleware (sandwich chika t'ant'a uñtasipawa), janiw anqäx t'ant'ar purinkasmati uka chikat jan pasaspa.

Middleware-ax mayiwinak uñchiqaspa, yant'aspa, qillqatanak mayjt'ayaspas jan ukax q'umachaspas, jan walt'irinakat jark'añataki.

Ukampis mayiwinakax walja guardianani chika utjaspas jan ukax jani, lurañ mayiwit uñtatjama, ukatx kawkhantix kutt'ayawit suyki ukhamarjamawa.

Ukhamatwa, sapa thakhi uñstawayt ukarux: Uñjama, khititix A punkur mayiw apani, q'illu thakhit jutpacha (GET), uka punkurux jan mayni guardiaski, ukhamax naya thakhimp churawayta (handler luriri mayiwi procesañataki).

Ukampis B punkur mayiw apanini, uka punkux quri jawillt'awi 'Wonka' aruni maypacha, ukhamax punkuchiri ucharayt yant'añataki (middleware). Punkuchirix pasañam amuypachaxa, B punkur mapamp churawayta (handler luriri procesaspa ukat kutt'ayätam).

Sapa handler mayiwinak aski yatiyawinak katuqaspa procesani, ukampis sapa kuti Response interface kutt'ayawinak kutt'ayapxani.

Jichhax kutt'ayawinaka extensibles utji, kunayman kutt'ayawi luraspa Response interface contrato phuqaspa.

Jan CPU ch'am ina ch'usar apnaqañataki, punkuchirix jan phuqhir uñjpachatamxa, jupanakax qhispiyätam, jan kutt'aspa chhaqhayawayätam. Mayiwix ukat thakhipax chhaqhawayapxi.

Ukatx amukt'ayañataki, servilleta arurjamax nayriri thakhinakawa:

1. Mayiwi (Request)
2. Front Controller
3. Thakhi apnaqawi (Método, Ruta, Handler luriri, Middleware llikachawi) (Route)
4. Kernel thakhinakat apnaqaspa mayiwi jikxatañataki. Middleware utji ukhaxa sapa kuti phuqaspawa. Taqini pasañ yatichpachaxa, phuqaskaspawa, ukampis mä jan walt'pachaxa, jark'awayapxi.
5. Mistiwi (Response)

Parina Framework ukaxa: mä aski thakhiwa, estricto, extensible, altamente yant'atäñapa ukatx lurañ apnaqirinakataki wali k'acha thakhi, sapa mayiwit handler handle lurawiki utjatapata.

Ukatpachas servilleta ukar uchapxi. P'iqin sinti phiskhukiwa, ukatx aski lurañ amtapachax, sapa kuti amtaruw purïta: machaq middleware uchañani, jupanakax jutirinak uñjirinakawa, jan walt'äwinak jark'irinakawa, sapa segundo mayiwinak jark'irinakawa, jan autorizat apnaqirinak khitirinakawa.

Uka jisk'a qillqatax yatichawayitu: request/response sinti jank'a, ukatx memory juk'aki apnaqäna.

Mä 2003 machinan 512MB RAM ukat Celeron apnaqirimpi, Parina framework apnaqaspa, sapa segundo 200 mayiwit kutt'ayäna, juk'a 40KB memory apnaqasa.

Ukatx amuyta mä wali qhananchat aski lurawi utjatayna. Juk'a recurso apnaqatayna ukatxa, juk'ampi khusa apnaqata jaqinakataki uñstatäspa, jan internet ch'aman jaqinakataki, jan ukax 15 mara machinak apnaqirinakataki.

Machaq lurawinak uñt'añatak alasiñamataki yatiwayta.

Mä q'axu jamp't'awi sur de Chile pachat!
