Tukuy imamanta ñawpaqtaqa, pampachayta mañakuni. Manam phiñachiyta munachkanichu, manam qamtachu niykiku rimaykunawan, chay hatun yachaysapa Veronica nisqan hina, ruranaykikunaqa allinmi kachkan.

Unay pacha qillqana ñawpaqpi codigokunata qillqaspa karqani. Llamkay pachaman lliqtaspa, allin ruraykunata hap'iyta munarqani, chay pacha rimaypa 10 classic pantayninkunata mana ruranapaq, chaynam unay pacha karqani.

Qunqaymanta web hatarichiypi llamkayta munarqani, ichaqa anchatam amirqani qillqanata sapa kuti takayta, chay yachaykunallata apnaqaspa hinaspa oficinamanta kaqllata ruraspalla. Huk machaq ñantam thaqharqani, mana sasa kaqta.

Jichhachqa, mañakuqqa katuqinmi: ancha allin ruranakunatam tarirqani, codigokuna ruraqkunam llamkayta ancha kusikunapaq rurarqaku, huk sumaq sintonía, huk nina k'anchay hina.

Chayraykum yachay wasiman kutiyta munarqani chay musuq miraykunaman yachachinaypaq kay milagroso ruranakuna utjakuyninta... walja kutitataqmi jawillt'aphuwanku... markaymanta hawamanpas.

Chaymanta castilla simi comunidadman chikañcharqani, Youtubeman wakin videokunata churaspalla, chaymanta pachata quykurqani musuq ruraqkunaman yachachinaypaq—khitinchus munayllawan programacionman yaykuqkunata, allichaykunata maskaqkunata kikin sasachakuyninkupaq.

Walja qillqasqakunata uka comunidadpaq rurarqani, ichaqa yatichiqpa munayninmi sapa kuti karqa: musuq miraykunaman yachachiyta munarqani imaynas llunk'utakuna rurakun, imaynas diseñakun, imas ukhupi kachkan.

Mana kikinchaptinchikqa, sapaqchallay thakhiyta hatarichirqani. Comunidadpi kapuq llunk'utakuna hina wakin llunk'utakunata rurayta munarqani, copia hina, ichaqa manam ancha allinchu karqa. Paykunaqa unay watakunata ñawpaqpi kachkarqaku.

Thakhi samarispa, huk punchaw modern PHP hinaspa SOLID patrónt'at yachaqayta alarqani, chaymanta arquitectura, CQRS patrónt'at qillqakunata qhawarqani, hinaspa ch'usaq qillqasqa (maintainable code).

Chaymanta aswan pisillawan llamkayta munarqani, huk ch'usaq yuyayman kutiyta, chaynam khitipas umallanman churayta atinman.

Tukuy imata llamkaspalla, umaypi k'anchay rikurirqa. Chaymanta yacharqani: tukuy rurasqanchikqa textota procesayllam. ¿Musuqchu? Unixmanta hamuptiykiqa taqinam yachanku. Ichaqa ñanpi puriq runakuna, machaq milagro yatiyawinaka katuqaqkuna... ¿chayta hap'inkuchu?

Chaynam huk framework llikachata huk jisk'a servilleta p'anqaman churayta munarqani.

Ichaqa manam anatt'anachu, Secure Code Warrior yachayqa ukhupimi chayaqpuhuarqa. Huk allin, seguro, yant'asqa (testeable) hinaspa ancha extensible uñt'atami.

Kayqa ancha jark'akuyniyuqmi jisk'a diagramapaq, ¿manachu?

Ñawpaq kaq, mayiwinaka chawpiyapachinapaq: front controller. Sumaq sutiyuqmi, ichaqa servidor kaqta sapa kuti kikin index.php qillqasqaman khitayllam.

Allinmi, tukuy puriykuna chawpiyachaqman hamunqa.

Iskay kaq, sapa mayiwim huk luraqllayuq kachkan (SOLID ukan S k'apak luraqnin), ukatx ancha pisiyachirqani: sapaq handler. Chayllam.

Kaypi samarisaq: handlerqa request katuqin, chaymanta procesan hinaspa response textota kutt'achin.

SOLIDpa huk amtanqa Interfaces apnaqaymi, chayqa mana imapaq plano hinam kaspapas, ichaqa contratos (métodos) kapuqninmi, hinaspa luraqkunaqa phuqhanankum (implementar).

¿Imapaqmi interfazqa? Pisillawan, ñuqapaqqa, kikinchasqa luraqkuna hatarichinapaqmi, ichaqa huk mayjt'asqa phuqhakuywan (polimorfismo).

¿Imatataq interfaces qunku? ¡Extensibilidad hinaspa estandarización sayachiyta!

Jichhachqa basicokunam utjanku. Ñankunata riqsichiq (Router). Mayiwinaka ñankunaman khitaq (Kernel).

Kunawsatix .NET, Java jan ukax JSP luratächi, Request hinaspa Response apnaqaspa, ukanax 'puriripa mayiwinapa' hinaspa 'kutt'ayapxatanas' ukhamawa.

¿Hinaspa seguridad?
Huk punchaw yacharqani: mayiw yaykunanpaqqa wakin jark'akunatam qhawananku. Palacio guardián hinam, software rimaypi, Middleware (sandwich chawpi aycha hina), mana chawpimanta pasaspaqa mana anqäx t'ant'aman chayanmanchu.

Middlewareqa mayiwta qhawan, yant'an, qillqakunata pichan, jan walt'irinakawan ch'aqwata jark'an.

Ichaqa mayiwikunaqa chawpipi walja guardianiyuq kanman, jan ukax manapas, mayiwpaman hina.

Chayrayku, sapa thakhita riqsichispay: Qhaway, khititix A punkuman mayiw apan, q'illu thakhimanta (GET), hinaspa punkupqa mana guardiayuq, ñuqa mapatam quykuni (handler uka mayiw procesanapaq).

Ichaqa, khititix B punkupaq mayiw apamun, punkutaq quri jawillt'awi Wonka aruyuq kaptin, porterota churaq kani (middleware). Portero pasayta niykuchaptinqa, mapatam quykuni (handler procesananpaq hinaspa imatapas kutt'achisunaykipaq).

Sapa handler mayiw yatiyawikunata katuqanqa, hinaspa sapa kuti Response interface contractowan hina kutt'aypuchanqa.

Jichhachqa Response kutt'aykuna extensibles kachkanku, Response interface contractowan hina.

Mana CPU ch'am chhaqtachinapaq, portero mana phuqhaqta qhawasuptyqa, chhaqtachisunkiñam, ñawpaqman mana qhawaspa. Mayiw hinaspa thakhinqa chhaqhakapunqam.

Qillqana rimayman hinaqa, kaykuna kikin thakhinkunam:

1. Mayiwi (Request)
2. Front Controller
3. Thakhi apnaqawi (Método, Ruta, Handler, Middleware llikachawi) (Route)
4. Kernel thakhinaka apnaqaspa mayiwi jikxatanapaq. Middleware kaptinqa sapa kuti phuqanqa. Taqini allin nispallaqa, phuqanqam, ichaqa huk mana allin kaptinqa, jark'awayapxi.
5. Mistiwi (Response)

Kayqa Parina Framework: huk allin thakhi, estricto, extensible, altamente yant'asqa hinaspa runapaq allin, sapa mayiwi handler handle luraqllayuq kaptin.

Chayraykum servilletaman churanqa. Ancha pisillam hap'inapaq, allin rurayta munaptiykiqa, kikin amtarum chayanki: machaq middleware uchananchik, paykunam puriykunata qhawaqkuna, jan walt'ayta jark'aqkuna.

Uka jisk'a qillqasqaqa anchatam yachachiwarqa: request/response ancha utqaylla, hinaspa memory pisi apnaqasqa.

Mä 2003 machinan 512MB RAM hinaspa Celeron apnaqirwan, Parina framework sapa segundo 200 mayiwikunata kutt'achirqa, pisi 40KB memory apnaqaspalla.
